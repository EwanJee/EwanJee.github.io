리멤버 프레디저 (경력/퇴직 진단 서비스)

  • Tech Stack: SpringBoot, JPA, JDBC, Thymeleaf, Mariadb, BootStrap
  • Website URL : Link
  • Github URL: Project Link

이 웹사이트는 커리어 컨설턴트 (카운슬러)를 대상으로 만들었습니다. 컨설턴트에게 경력/퇴직 진단 링크가 주어지고, 컨설턴트는 고객들에게 링크를 보내 진단 테스트를 받게하고, 웹 사이트내에서 진단 결과를 확인하게 해주는 서비스입니다.

데이터베이스를 바탕으로 로그인, 회원가입 기능, 진단테스트를 구현하였고, Spring MVC 를 이용하였습니다.
진단 테스트는 5가지의 질문을 통해 진단 결과를 도출하고, 결과는 그래프로 시각화하여 보여줍니다.

아쉬운 점:
- 사용자들의 편리성을 위해서는 진단 링크를 카카오톡이나 이메일로 자동으로 보내질 수 있는 API를 이용하고 싶었으나 더 공부가 필요하다고 판단하여, 직접 링크를 복사하여 보내야하는 점이 아쉽습니다.
- JPA 환경에서 JDBC를 썼는데 JPA를 쓰는 방법도 있지 않을까 하는 생각이 들었습니다.
- 진단 결과를 pdf가 아닌 html로 했는데, pdf로 export하는 방법을 찾고 있습니다.
- Spring Security 공부를 해서 적용을 해야겠습니다.
