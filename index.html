How is a website created?
1. A planner is needed (website structure, features, design)
2. UX/UI design is required (website design, icons, buttons, PDF)
3. A frontend developer is needed (implementing website design into code)
4. A backend developer is needed (implementing website functionality -- business logic, etc.)
5. A database administrator is needed (managing the website's data)
6. A web hosting manager is needed (making the website publicly accessible on the internet)
7. For the ongoing maintenance of the web, a system administrator is needed (website security, backups, updates)
